# 운동 분석 시뮬레이터

브라우저에서 실행되는 물리 운동 분석 및 가속도 모델 추정용 클라이언트 앱입니다.
내장 데이터셋 관리, 파생 변수 계산, 그래프 표시, OLS, Lasso 피팅, 신경망 학습을 모두 브라우저 안에서 실행합니다.

## 설치

```bash
npm install
```

## 실행

```bash
npm run dev
```

개발 서버가 출력하는 로컬 주소를 Google Chrome에서 열어 확인합니다.

## 빌드

```bash
npm run build
```

## 참고

- 이 앱은 클라이언트 사이드 전용입니다.
- 백엔드, 데이터베이스, 서버 사이드 계산은 사용하지 않습니다.
- 수치 계산은 브라우저 내부 TypeScript 코드로 실행됩니다.
- 내장 원본 데이터는 `t`, `x`로 구성되며, 앱에서 `v`, `a`를 계산합니다.

## 신경망 학습 + Symbolic Distillation

`신경망 학습` 탭은 선택된 데이터셋의 측정 가속도 `a`를 목표값으로 두고, 선택한 입력 변수 `x`, `v`, `t`로 작은 완전연결 신경망을 브라우저에서 학습합니다.

학습 후에는 신경망 예측값 `â_NN`을 새로운 목표값으로 사용하여 Lasso 증류를 수행합니다. 즉, 일반 Lasso가 측정 가속도 `a`를 직접 맞추는 것과 달리, 이 흐름은 신경망이 학습한 예측 행동을 학생이 입력한 후보 항 조합으로 근사합니다.

후보 항은 기존 안전 수식 파서와 같은 문법을 사용합니다.

- 변수: `t`, `x`, `v`
- 연산자: `+`, `-`, `*`, `/`, `^`
- 함수: `abs()`, `sin()`, `cos()`, `exp()`, `log()`, `sqrt()`

추출된 식은 교육용 근사식이며, 항상 실제 물리 법칙을 복원한다고 보장하지 않습니다.
