# 운동 분석 시뮬레이터

브라우저에서 실행되는 물리 운동 분석 및 힘 법칙 추정용 클라이언트 앱입니다.
현재 단계는 Step 0 프로젝트 기반 구축이며, 실제 수치 계산 기능은 포함하지 않습니다.

## 설치

```bash
npm install
```

## 실행

```bash
npm run dev
```

개발 서버가 출력하는 로컬 주소를 Google Chrome에서 열어 확인합니다.

## 빌드

```bash
npm run build
```

## 참고

- 이 앱은 클라이언트 사이드 전용입니다.
- 백엔드, 데이터베이스, 서버 사이드 계산은 사용하지 않습니다.
- 이후 수치 계산 기능도 브라우저 내부에서 실행될 예정입니다.
- 현재는 레이아웃, 보기 전환, 클라이언트 사이드 데이터셋 선택 및 CSV 가져오기만 구현되어 있습니다.
- CSV 파일은 브라우저에서만 파싱되며 `t`, `x`, `v`, `a` 열이 필요합니다.
